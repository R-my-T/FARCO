Instructions to upload the department's people's information:

  1. Create a csv file in the format: Unique_ID*, Name_of_Person, Contact_email

  2. Upload the csv file

  3. Upload the images by choosing them in the same order as the csv file i.e, first person's image should be selected first and so on.

  4. Kindly ensure that the images are in jepg,jpg or png format.

  5. Click on submit.

* Unique ID should be a distinctive ID that the person is recognised with. For instance: registration number in case of students, employee ID in case of employees etc.
